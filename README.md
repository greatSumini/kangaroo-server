## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# re-build docker image
$ npm run dev:build

# remove docker container
$ npm run dev:down
```

## Test

```bash
# unit tests
$ npm run jest

# e2e tests
$ npm run jest:e2e

# test coverage
$ npm run jest:cov
```

## Deploy

```bash
# run run.sh
$ ./run.sh
```
